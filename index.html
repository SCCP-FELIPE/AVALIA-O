Resumo: Tabela periodica dos Elementos

Introdução

A tabela peridodica é uma ferramenta fundamental para entender a organização dos elementos quimicos. Ela foi proposta por Dmitri Medleev

clasificação dos Elementos

  • metais: sao bons vondutures de calor e eletrecidade.
  • ametais: possuem propiedades opostas aos metais.
  • Gases nobres: são quimicamente estáveis.
  • semimetais: apresentam propiedades intermediarias.

Propiedades importantes

  1. Número atômico: quantidade de prótens no núcleo.
  2. Massa atômica: soma dos prótons e nêutrons.
  3. Eletronegatividade: tendência de um âtomo atrair elétrons.
